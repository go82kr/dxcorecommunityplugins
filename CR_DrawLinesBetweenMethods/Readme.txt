= Draw Lines Between Methods DxCore Add-in =

This is a CodeRush add-in that draws line between methods, property procedures, and classes: a bit like how the Visual Basic editor does.

The lines can be customised using the settings page:

== Installation ==
Download, unzip and copy the DLL to \Program Files\Developer Express Inc\DXCore for Visual Studio .NET\2.0\Bin\PlugIns\, then restart Visual Studio.
